# FilmDate - 复古胶片情侣创意照片平台设计文档

## [S1] 项目概述

**项目名称**：FilmDate

**核心理念**：一个复古胶片风格的情侣创意照片平台，通过"每日主题挑战"和"自由拼贴编辑"两种模式，让情侣用照片记录和创造共同回忆。

**目标用户**：情侣（两人一组）

**核心功能**：
1. 每日主题挑战 - 每天一个主题，两人分别拍照回应
2. 复古胶片拼贴编辑器 - 自由创作拼贴作品
3. 时间线回顾 - 按时间浏览所有作品

## [S2] 每日主题挑战系统

**主题库**：
- 预设 100+ 个主题，分为不同类别：
  - 情感类："此刻的心情"、"今天最开心的事"
  - 观察类："窗外的风景"、"今天的天气"
  - 创意类："用红色拍一张"、"仰拍视角"
  - 生活类："今天的早餐"、"手边的物品"

**挑战流程**：
1. 每天 0:00 系统自动推送当日主题
2. 两人各自用手机/电脑拍照或选择相册照片
3. 上传后，系统自动应用复古胶片滤镜
4. 两张照片并排显示，形成"情侣视角对照"
5. 可以添加简短文字说明

**特殊机制**：
- 连续打卡奖励（7天、30天、100天）
- 偶尔出现"双人合作"主题（如拼成一张完整的图）
- 历史挑战回顾功能

## [S3] 复古胶片拼贴编辑器

**画布系统**：
- 基于 Canvas API 的可视化编辑器
- 支持多种画布尺寸（1:1、4:3、16:9、自由尺寸）
- 撤销/重做功能

**素材库**：
- 复古胶片滤镜（10+ 种，模拟不同胶片效果）
- 复古贴纸（相机、胶卷、日期戳、天气图标等）
- 文字工具（手写体、打字机字体）
- 背景纹理（胶片颗粒、划痕、光晕）

**拼贴功能**：
- 照片拖拽定位、缩放、旋转
- 多图层支持（置顶/置底）
- 对齐辅助线
- 透明度调节

**导出**：
- 高清 PNG 导出
- 可选是否显示复古边框
- 保存到时间线

## [S4] 数据模型与技术架构

**数据模型**：
```
User（用户）
├── id, name, avatar
├── couple_id（关联情侣关系）
└── created_at

Couple（情侣关系）
├── id
├── user1_id, user2_id
├── invite_code（邀请码，用于配对）
└── created_at

DailyChallenge（每日挑战）
├── id
├── date
├── theme_id（关联主题）
└── status（进行中/已完成）

ChallengeResponse（挑战回应）
├── id
├── challenge_id
├── user_id
├── photo_url
├── caption（文字说明）
├── filter_id（使用的滤镜）
└── created_at

Collage（拼贴作品）
├── id
├── couple_id
├── title
├── canvas_data（JSON，存储画布状态）
├── thumbnail_url
└── created_at

Theme（主题库）
├── id
├── content（主题内容）
├── category（类别）
└── is_special（是否特殊主题）
```

**技术栈**：
- 前端：Next.js 14 + TypeScript + Tailwind CSS
- 画布：Fabric.js（成熟稳定的 Canvas 库）
- 后端：Next.js API Routes
- 数据库：Supabase（PostgreSQL + 实时订阅）
- 存储：Supabase Storage（照片存储）
- 认证：Supabase Auth（邮箱/手机号登录）

**部署**：Vercel（前端）+ Supabase（后端服务）

## [S5] UI/UX 设计

**视觉风格 - "复古胶片博物馆"**：
- 全屏沉浸式背景，模拟暗房/胶片放映氛围
- 动态胶片颗粒效果（CSS 动画）
- 照片展示像实体照片一样"漂浮"在页面上
- 悬停时有胶片卷动、光影变化效果

**首页 - "回忆墙"**：
- 全屏瀑布流布局，照片像拍立得一样"散落"在桌面上
- 每张照片有手写体标注、日期戳
- 鼠标悬停时照片轻微旋转、放大
- 点击后像翻转照片一样查看背面的故事
- 背景是缓慢变化的暖色渐变

**每日挑战 - "放映厅"**：
- 进入时有胶片放映机动画
- 今日主题以霓虹灯风格大字显示
- 两个照片区域像电影银幕一样展开
- 上传照片时有"咔嚓"快门音效
- 完成挑战后有胶片卷动庆祝动画

**拼贴编辑器 - "暗房工作室"**：
- 背景模拟暗房红色安全灯氛围
- 工具栏像实体工具一样"摆放"在工作台上
- 拖拽素材时有物理惯性效果
- 画布边缘有胶片齿孔装饰
- 保存时有"显影"动画效果

**交互动效**：
- 页面切换：胶片卷动过渡
- 按钮点击：复古快门反馈
- 加载状态：胶片卷动进度条
- 成就达成：放映机聚光灯效果

**配色方案**：
- 主色调：深棕 (#2C1810) + 米黄 (#F5E6D3)
- 点缀色：暗红 (#8B2500) + 金色 (#D4AF37)
- 强调色：霓虹粉 (#FF6B9D) 用于特殊元素

## [S8] 技术实现细节

**滤镜引擎实现**：
- 使用 Canvas API 的像素操作实现滤镜
- 预设滤镜参数矩阵（色彩映射、对比度、饱和度）
- 支持滤镜叠加和强度调节
- 胶片颗粒效果：随机噪点 + 混合模式

**Fabric.js 编辑器优化**：
- 自定义对象类型（照片、贴纸、文字）
- 性能优化：对象缓存、脏区域重绘
- 移动端触摸支持
- 快捷键支持

**实时功能**：
- Supabase Realtime 订阅挑战状态
- 上传照片时实时通知对方
- 协作编辑时的光标同步

**性能优化**：
- 图片懒加载 + 渐进式加载
- Canvas 渲染优化（离屏渲染、对象池）
- 照片压缩（上传前客户端压缩）
- CDN 加速（Vercel Edge Network）

**安全考虑**：
- 照片存储访问控制（仅情侣可见）
- 上传文件类型验证
- 图片内容安全检查（可选）
- 数据备份策略
